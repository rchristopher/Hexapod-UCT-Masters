◆ processAccelData()

void Accelerometer::processAccelData ( void  )

Process the accelerometer data once it has been received.

Process the incoming accelerometer data, pulling out the various bits of data and combining them into pitch, roll and yaw. Only if the checksum is correct meaning that the data has not been corrupted.
